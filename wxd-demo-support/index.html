    • Streamlit - Web interface framework
    • Llama Index - Data framework for building LLM applications
    • pyMilvus - Python SDK of Milvus

    • easyocr - OCR reader which scans images and extracts text in 80+ languages
    • pymupdf - PyMuPDF is a high performance Python library for data extraction, analysis, conversion & manipulation of PDF documents
